# Model SDK Documentation

This directory contains documentation and examples for all supported AI model SDKs.

## Directory Structure

```
docs/
├── claude4_5/
│   ├── docs/
│   ├── examples/
│   ├── readme.python.md
│   ├── readme.typescript.md
│   └── README.md
├── gemini3/
│   ├── docs/
│   ├── readme.python.md
│   ├── readme.typescript.md
│   └── README.md
└── gpt5_2/
    ├── docs/
    ├── examples/
    ├── readme.python.md
    ├── readme.typescript.md
    └── README.md
```

## Available Models

To use a specific model, please refer to its dedicated README:

- **[Claude 4.5](./claude4_5/README.md)** - Anthropic's Claude 4.5 API documentation and examples
- **[Gemini 3](./gemini3/README.md)** - Google's Gemini 3 API documentation and examples
- **[GPT-5.2](./gpt5_2/README.md)** - OpenAI's GPT-5.2 API documentation and examples

Each model directory contains:
- `docs/` - Detailed documentation for the model's features and capabilities
- `examples/` - Code examples demonstrating usage (Python and TypeScript)
- `readme.python.md` - Python-specific usage guide
- `readme.typescript.md` - TypeScript-specific usage guide
