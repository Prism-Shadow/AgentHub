# AgentAdapter Documentation

This directory contains reference documentation for coding agents and developers working on the AgentAdapter project.

## Available Documentation

- [Architecture Guide](./architecture.md) - System architecture and design decisions
- [Development Guide](./development.md) - Development workflows, build processes, and testing guidelines
- [Code Conventions](./conventions.md) - Coding standards and best practices for both Python and TypeScript

## Quick Start

For developers new to this project:

1. Read the [Architecture Guide](./architecture.md) to understand the project structure
2. Review [Development Guide](./development.md) for setup and workflow instructions
3. Familiarize yourself with [Code Conventions](./conventions.md) before making changes

## Purpose

This documentation serves as a reference for:
- AI coding agents (like GitHub Copilot) to better understand the project
- Human developers onboarding to the project
- Maintaining consistency across contributions
- Documenting architectural decisions and rationale
