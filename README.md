# Agent Adapter

This repository contains a minimal Python package and a minimal TypeScript package for the Agent Adapter.

## Python package

Python sources live in `src_py/`.

```bash
cd src_py
make
```

See [docs/auto_llm_client.python.md](docs/auto_llm_client.python.md) for comprehensive usage examples and API documentation.

## TypeScript package

TypeScript sources live in `src_ts/`.

```bash
cd src_ts
make
npm run start
```
